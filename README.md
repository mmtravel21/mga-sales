# 🍡 명가삼대떡집 판매 분석 대시보드

이지어드민에서 받은 일자별 주문 파일을 업로드하면 자동으로
일일 보고서 + 월별 누계 + 마진 + 캘린더를 생성합니다.

---

## 🚀 최초 설치 (1회만)

### 1️⃣ Python 설치
https://www.python.org/downloads/ → Python 3.10 이상 다운로드
- ⚠️ 설치 시 **"Add Python to PATH"** 옵션 반드시 체크

### 2️⃣ 라이브러리 설치
폴더에서 `setup.bat` **더블클릭**

### 3️⃣ (선택) 자동 시작 등록
`install_autostart.bat` 더블클릭 → Windows 켜질 때 자동 실행되도록 등록

---

## ▶️ 실행

`run.bat` 더블클릭 → 자동으로 서버 시작 + IP 표시

### 접속 주소
| 누가 | 주소 |
|---|---|
| 본인 (이 PC) | http://localhost:8501 |
| 같은 사무실 다른 PC | http://(서버 PC의 IP):8501 |
| 모바일 (같은 와이파이) | http://(서버 PC의 IP):8501 |

run.bat 실행하면 콘솔에 IP 가 표시됩니다.

---

## 📋 일일 사용법 (담당자)

1. **이지어드민**에서 `확장주문검색_*.xls` 다운로드
2. 브라우저에서 http://localhost:8501 접속
3. 좌측 사이드바에 파일 **끌어다 놓기**
4. 자동 감지된 **기준일(송장입력일)** 확인
5. **💾 DB에 저장** 클릭
6. **📊 일일 종합 보고서** 탭 → 화면 캡쳐 (`Win+Shift+S`) → 대표님께 보고

### 같은 관리번호가 다른 날짜로 잘못 들어간 경우
업로드 시 **🔄 "기존 데이터 기준일 갱신 (UPSERT)"** 체크 → 새 파일 기준으로 덮어쓰기

---

## 💾 데이터 보호 (중요)

### 자동 백업
- **저장/초기화 직전마다** `backup/` 폴더에 `sales_YYYYMMDD_HHMMSS.db` 자동 생성
- 최근 30개 백업본 유지 (오래된 것 자동 삭제)
- OneDrive 폴더 안에 있으면 클라우드까지 자동 동기화 ✅

### 백업에서 복구
사이드바 → ⚙️ 데이터 관리 → 📦 백업 파일 선택 → ⏪ 선택 백업으로 복구

### DB 초기화 (위험)
사이드바 → ⚙️ 데이터 관리 → **"RESET"** 입력 → 🗑️ DB 전체 초기화
(초기화 직전에도 백업이 자동 생성되어 복구 가능)

## ⚙️ 설정 (사이드바)

### 📝 전월 평균 수동 입력
DB에 전월 데이터 누적되기 전까지 임시로 수기 입력
- 보고월 입력 (예: 2026-05)
- 4개 카테고리 × 3개 값 (평균/월요일/월제외) 입력
- DB에 데이터 쌓이면 자동으로 우선 적용

### ⚙️ 월 목표 설정
- 넥스트고 전체 (OEM포함): 기본 ₩850,000,000
- 넥스트고 생산만: 기본 ₩800,000,000
- 오픈마켓 (OEM포함): 기본 ₩200,000,000
- 오픈마켓 생산: 기본 ₩190,000,000
- 박스당 택배비: 기본 ₩3,000

---

## 📐 계산 공식 (시트 동일)

```
넥스트고 전체 매출 = 자사 원가 + 자사 택배비 + 오픈마켓 OEM포함 정산금
넥스트고 생산 매출 = 생산 원가 + 생산 택배비 + 오픈마켓 생산 정산금
오픈마켓 OEM포함 매출 = 오픈마켓 + 모든 공급처 정산금
오픈마켓 생산 매출 = 오픈마켓 + 넥스트고만 정산금

오픈 마진 = 매출 - 원가 - 택배비
```

### 자동 보정
- 같은 상품코드의 단가가 1건만 비정상 (2배 이상/0.5배 이하) → 자동으로 정상 단가로 보정
- 파일 내부 관리번호 중복 → 첫 행만 자동 유지

## ❓ 문제 해결

| 증상 | 원인 / 해결 |
|---|---|
| 캘린더에 12일 등이 누락으로 표시 | 5/13 파일에 묶여서 들어간 경우 → 5/12 파일 다시 업로드 시 **UPSERT 체크** |
| 파일 업로드 실패 (Execution failed) | 파일 내부 중복일 가능성 → 이미 자동 처리됨 |
| 사이드바 ⚙️ 안 보임 | 사이드바 폭이 너무 좁음. 사이드바 경계 드래그해서 넓혀주세요 |
| 다른 PC에서 접속 안 됨 | 윈도우 방화벽 → "Python" 또는 "Streamlit" 허용 |
| Python 인식 안 됨 | Python 재설치 시 "Add to PATH" 체크 |
